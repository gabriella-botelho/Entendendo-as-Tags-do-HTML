Tags de formatação

Voltar

Essas são algumas tags para a formatação de texto, por exemplo:

  • Tag "b" serve para deixar o texto em negrito
  • Tag "em" serve para enfatizar o texto
  • Tag "i" serve para deixar o texto em itálico
  • Tag "small" serve para deixar o texto pequeno
  • Tag "sub" serve para deixar o texto subscrito
  • Tag "sup" serve para deixar o texto sobrescrito
  • Tag "strong" serve para deixar demonstrar a importância do texto
  • Tag "mark" serve para destacar palavras-chaves em um resumo
  • Tag "u" serve par sublinhar palavras
  • Tag "blockquote" serve para
    fazer uma sitação
    no texto
  • Tag "abbr" serve para idicar uma abreviação ou sigla


Tags de paragrafos

Voltar

Essas são algumas das tags de para formatação de paragrafos

  • Tags "h1, h2, h3, h4, h5 e h6" são tags utilizadas para a criação de títulos e cabeçalhos, cada número tem um tamanho de texto diferente sendo o h1 o maior e o h6 o menor
  • Tag "p" é utilizada para a criação de paragrafos
  • Tag "br" é usada para pular uma linha entre as tags
  • Tag "hr" é usada para colocar uma linha de separação entre as tags


Tags de listas

Voltar

Essas são algumas das tags para listas ordenadas e não ordenadas:

  • Tag "ol" serve para criar uma lista ordenada
  • Tag "li" dentro da tag "ol" serve para ordenar uma lista com números, letras etc
  • Tag "ul" serve para a criação de listas não ordenadas
  • Tag "li" dentro da tag "ul" serve como marcadores


Tags de tabelas

Voltar

Essas são algumas das tags para tabelas:

  • Tag "tr" serve para colocar linha nas tabelas
  • Tag "td" serve para colocar coluna nas tabelas
  • Tag "table" serve para colocar estrutura na tabela
  • Tag "th" serve para colocar um cabeçalho na tabela
  • O "colspan" é um atributo colocado na tag "th" para mesclar as células das linhas da tabela
  • O "rowspan" é um atributo colocado na tag "th" para mesclar as células das colunas da tabela
  • O " " é um atributo que serve para preencher um espaço vazio de uma tag
